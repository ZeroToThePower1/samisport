Math Quiz Game (Console-based)

This Python script runs an infinite arithmetic quiz in the console, where the user is repeatedly asked to solve randomly generated math problems involving addition (+), subtraction (-), or multiplication (*). It works like this:

Main Flow:

Question Generation:

If the previous input was valid (answered = True), the program generates a new math problem using two random numbers between 1 and 100 and a random operator from +, -, or *.

User Input:

It displays the equation (e.g., 45 + 17 = ?) and asks the user to type in the answer.

Answer Checking:

The user’s input is converted to a float.If it matches the correct result, the program: Prints “Correct” Increases the score by 1 If incorrect, it shows the correct answer.

Error Handling:

If the input is not a valid number (like letters or blank), the program prints an error and repeats the same question.

Score Tracking:

The user’s score is printed after each correct attempt and accumulates over time.

Features:

Infinite loop (while True) for continuous play.Error handling using try-except to deal with invalid inputs.Score counter to track the number of correct answers.Keeps asking the same question if invalid input is entered.
